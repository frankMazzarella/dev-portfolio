Some of my favorite tools

Docker is a container system that provides OS level virtualization. This allows developers to build applications independently to the variations of hardware and always run the same.
Express has earned its reputation as the de facto standard for serving web apps and API's using Node. Express helps developers organize their code into an MVC structure on the back-end.
Server-side JavaScript helps minimize developer fatigue by keeping the tooling familiar throughout the stack while managing to provide blazing fast API response times.
A time proven and familiar approach to managing relational data. MySQL is one of the most widely used open-source DBMS solutions available.
Combining the power of Angular and Cordova to build apps, Ionic alleviates the hard work of mobile development by delivering native-feeling experiences across both iOS and Android using standard web technologies.
Angular is a feature rich framework for building large-scale enterprise-grade single page applications. Users will receive a highly-optimized bundle that provides dynamic loading for native feeling interfaces.
JSON Web Tokens are a standard that describe how to securely transfer data over the web. Tokens are signed by the server making them a great way to securely deliver information and represent authentication.
TypeScript is a superset of JavaScript that adds rigidity by including support for optional types, classes, and interfaces. It transpiles into vanilla JS that can be used anywhere, both client side and on the server.
